== Description ==

* En
This plugins add two new fields to formidable. With one you generate and random string, with the other you can validate if the given string exist in other form.

Example:
You need to send and security key to the user, and validate in other form if the key is valid. Then you create two form one to generate the random string and other to validate.
In the first form you need to configure the field generator with the length of the string and if you grant special chars.
In the second form you need to add the validation field and select the form target, and the validation type.

Enjoy!

* Es
El plugins adiciona dos campos a formidable. El primero es para generar una cadena de texto aleatoria y el segundo es para validarla desde otro formulario.

Ejemplo:
Se necesita enviar un código de seguridad, y validarlo desde otro formulario. Lo que necesitas hacer es crear dos formularios, uno para generar las llaves y el otro para validarlas.
En el primero formulario adicionas el campo de generar la llave, y configuras el tamaño y si permitirás caracteres especiales.
En el segundo adicionas el campo de validación, luego configuras la fuente desde el cual verificaras las cadenas y el tipo de validación.
